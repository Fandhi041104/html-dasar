Uji Coba 4




Atribut type="button" digunakan untuk membuat tombol yang dapat di klik dan melakukan tindakan khusus saat ditekan. ketika button "Click Me!" di tekan, maka akan muncul papan peringatan/pengumuman yang bertuliskan Hello World! pada bagian atas.

Uji Coba 5




- type="datetime-local" digunakan untuk membuat kolom input user yang memungkinkan user untuk memilih tanggal dan waktu.
- type="month" digunakan untuk membuat kolom input bagi user untuk memilih month/bulan.
- type="email" digunakan untuk mengumpulkan alamat email dari user.

Uji Coba 6




- type="image" digunakan untuk membuat tombol kirim grafis.
- type="file" digunakan untuk membuat kolom input untuk memilih direktori atau file lokal untuk diunggah ke server.
- type="hidden" digunakan untuk membuat kolom input yang tidak ditampilkan kepada user, namun disertakan dalam data form saat form dikirimkan.

Uji Coba 7



- type="quantity" digunakan untuk membuat kolom input yang memungkinkan user untuk memasukkan suatu besaran atau satuan yang berupa angka.
- type="range" digunakan untuk membuat kontrol untuk memasukkan angka dalam range/rentang tertentu.

Uji Coba 8




- type="url" digunakan untuk membuat kolom input untuk memasukkan sebuah url.
- type="search" digunakan untuk membuat kolom input untuk permintaan pencarian.
- type="tel" digunakan untuk membuat kolom input nomor telepon, dan dirancang untuk menerima masukan yang cocok dengan pola nomor telepon yang telah di setting.

Uji Coba 9



- type="time" digunakan untuk membuat kolom input untuk memasukkan waktu, dan dirancang untuk menerima input dalam format jam, menit, dan detik.
- type="week" digunakan untuk membuat kolom input bagi user untuk memasukkan week/minggu dalam satu tahun
